
# GerenciaLivros 

Sistema para gerenciar os livros que foram emprestados 


## Levantamento de requisitos 

### Requisitos funcionais 

O sistema deve ter: 

- Cadastro de livros 
- Listagem livros
- Filtragem de livros 
- Pesquisa de livros
- Visualização de emprestimos de livros  
- Atualização emprestimos
- Geração de relatórios de emprestimos dos livros  

Estoria | Descrição 
:--------:|----------
EU1 | Como usuário devo cadastrar os livros que possuo. 
EU2 | Como usuário devo visualizar os livros que possuo.
EU3 | Como usuário preciso buscar por um livro que está cadastrado no sistema.
EU4 | Como usuário preciso visualizar as informações relacionadas ao emprestimo dos livros que possuo.  
EU5 | Como usuário preciso gerar um relátorio dos livros que possuo.

### Requisitos não funcionais 

O sistema deve ser:

- Responsível, com um tempo de resposta aceitável 
- Fácil de mecher, deve ter um interface simples 
- Simples de entender
